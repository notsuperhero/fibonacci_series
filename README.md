# Fibonacci Visualizer

A React application to visualize the Fibonacci sequence.

## Features
- Input to control the number of terms.
- Auto-play mode.
- Visual representation using a logarithmic scale graph.
- List view of the generated sequence.

## Technologies
- React
- Vite
- Framer Motion

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Run the dev server:
   ```
   npm run dev
   ```
